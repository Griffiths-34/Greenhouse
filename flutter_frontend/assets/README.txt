INSTRUCTIONS TO SET UP APP ICON:
================================

1. Save your logo image (the green leaf with blue waveform) to this folder as:
   app_icon.png

2. Image requirements:
   - Size: 1024x1024 pixels (minimum)
   - Format: PNG
   - Square dimensions
   - Transparent or solid background

3. After saving the image, run this command in the flutter_frontend directory:
   
   flutter pub run flutter_launcher_icons

4. This will automatically generate icons for:
   ✓ Android (all sizes)
   ✓ iOS (all sizes)
   ✓ Web
   ✓ Windows
   ✓ macOS
   ✓ Linux

5. Then rebuild your app to see the new icon!

Your current configuration (in pubspec.yaml):
- Android: ✓ Enabled
- iOS: ✓ Enabled
- Web: ✓ Enabled
- Windows: ✓ Enabled
- macOS: ✓ Enabled
